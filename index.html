Run It With Your Own n8n Docker

Each developer keeps their own OpenAI and CallMeBot keys in local environment variables. The workflow JSON stays safe to share.

Step 1

Start n8n

Run n8n locally with Docker and pass in your `.env` file.

Step 2

Import workflow

Open `localhost:5678`, import the JSON, then activate it.

Step 3

Open the board

Execute from Schedule Trigger once, then let the active workflow refresh daily.

cp .env.example .env
docker run --rm -it \
  --name n8n-launch-board \
  -p 5678:5678 \
  --env-file .env \
  -v n8n_launch_board_data:/home/node/.n8n \
  docker.n8n.io/n8nio/n8n
